The TOROS LVC GCN alert responder
=================================

This is the documentation for the `TOROS`_ responder to the Gamma- ray burst Coordinates Network (`GCN`_)
for Gravitational-Wave (GW) alerts from the LIGO-Virgo Collaboration (`LVC`_).

The responder lvcgcnd is a `daemon`_ managed by the `systemd`_ service found in most Linux systems.
