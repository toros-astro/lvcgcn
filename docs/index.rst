The TOROS LVC GCN alert responder
=================================

This is the documentation for the TOROS responder to the Gamma-ray burst Coordinates Network (GCN)
for Gravitational-Wave (GW) alerts from the LIGO-Virgo Collaboration (LVC).

The responder lvcgcnd is a daemon managed by the systemd service found in most Linux systems.
